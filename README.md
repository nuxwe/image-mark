# image-mark-mobile

#### 介绍
一款移动端图片标注处理方案


#### 使用说明
```
import ImageMark from "image-mark-mobile";

var Mark=new ImageMark(); 
```
### Event
|事件|说明|值|
|---|---|---|
|init|初始化图片|el\url\|
|clearAll|清除画布上的所有元素|-|
|clearLast|撤销上一次标注|-|
|arc|开始画圆|-|
|line|开始画带箭头线|-|
|getImageList|返回效果图|callback|
