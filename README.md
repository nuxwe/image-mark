# image-mark-mobile

#### 介绍
一款移动端图片标注处理方案


#### 使用说明
```
import ImageMark from "image-mark-mobile";

var a1=new ImageMark(); 

a1.init();//初始化el url 

a1.clearAll();//清除画布 

a1.clearLast();//撤销上一次标注 

a1.arc();//画圆 

a1.line();//画带箭头的线 

a1.getImageList();//输出多种图片 
```
